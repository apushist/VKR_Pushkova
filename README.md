# VKR_Pushkova
> Код системы в файле interface.ipynb
### Цель и задачи
**Целью данной работы** являлась разработка интеллектуальной системы на основе нейронных сетей, способной распознавать нотные партитуры для гитары по изображениям (скан-копия, фотография) и формировать для каждой ноты указания по выбору оптимальных струны и позиции на грифе, то есть аппликатуру для прочитанных партитур.

**Задачи:**
1. Анализ существующих методов для распознавания нотных партитур по изображениям.
2. Разработка оптимальной архитектуры системы, сочетающей детерминированные алгоритмы с нейронными сетями.
3. Реализация детерминированного алгоритма для определения позиций нотных линий на изображении.
4. Поиск подходящего набора данных и его предобработка для работы с выбранной архитектурой нейронной сети.
5. Обучение модели нейронной сети для распознавания музыкальных символов на изображении.
6. Разработка алгоритма для объединения данных, извлеченных из изображения вышеописанными методами, и формирования списка нот.
7. Разработка подсистемы, преобразующей сформированный список нот в аппликатуру для классической шестиструнной гитары.
